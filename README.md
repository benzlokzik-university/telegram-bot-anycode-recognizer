## Getting started

```shell
cd telegram-bot-anycode-recognizer
```
<br/>

#### Creating venv
```shell
pip -m venv venv
```

<br/>

#### Activating venv
Unix-like os
```shell
source ./venv/bin/activate
```
Windows
```shell
./venv/Scripts/activate.bat
```

<br/>

#### Installing requirements from ```requirements.txt``` 
```shell
pip install -r requirements.txt
```

<br/>

---


## Running python code

```shell
python main.py
```
